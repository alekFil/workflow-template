Add a new ADR to `.context/decisions.md`.

Use the next sequential number after the last ADR in the file.

## Format

```markdown
## ADR-{N}: {Title}

**Status:** Accepted

**Context:** {what prompted the decision — problem or situation}

**Decision:** {what was decided and why}

**Consequences:** {what changes as a result}
```

Prepend the new ADR at the top of the file, above the existing entries.
