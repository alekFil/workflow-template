Шаблон рабочего процесса для проектов на [Claude Code](https://claude.ai/code).

Клонируй, запусти скрипт — и новый проект готов к работе с CC: документация, мета-скиллы и рабочий процесс уже настроены.

---

## Быстрый старт

```bash
git clone <url> <название-проекта>
cd <название-проекта>
./scripts/init-project.sh
```

Подробнее — в [SETUP.md](SETUP.md).

---

## Что входит

- **`CLAUDE.md`** — инструкции для CC: роли, ключевые фразы, конвенции
- **`WORKFLOW.md`** — шпаргалка по рабочему процессу
- **`.claude/skills/meta/`** — мета-скиллы: фиксируем, закрываем задачу, текущий статус, синхронизируем, экспорт истории
- **`docs/`** — структура документации: blueprint, plan, to-do, status, decisions

---

## Философия

Этот шаблон — не попытка автоматизировать разработку. Полная автоматизация нетривиального проекта — это другой класс задач.

Цель здесь другая: качественное human-AI сотрудничество. Workflow добавляет checkpoint'ы, а не убирает их — явное разделение на фазы обсуждения, проектирования и реализации не позволяет CC уйти не туда без ведома разработчика.

Реальный результат — снизить cognitive overhead и сделать работу с CC предсказуемой. Не нужно каждый раз объяснять контекст, договариваться о стиле работы, помнить где что лежит. Это личная рабочая система, и она работает для людей с похожим стилем работы.
